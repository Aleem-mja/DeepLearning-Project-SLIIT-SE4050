# 🎬 IMDB Movie Review Sentiment Analysis using RNN  
### Deep Learning Project — SLIIT SE4050  

---

## 📖 Project Overview
This project focuses on **sentiment analysis** of IMDB movie reviews using a **Recurrent Neural Network (RNN)**.  
The model classifies reviews as **positive** or **negative**, providing valuable insights into audience opinions and emotions through Natural Language Processing (NLP).

---

## 📊 Dataset Information
- **Dataset Name:** IMDB Movie Reviews  
- **Source:** [Stanford AI Lab — IMDB Dataset](https://ai.stanford.edu/~amaas/data/sentiment/)  
- **Description:**  
  This dataset contains **50,000** movie reviews from IMDB, labeled by sentiment (positive or negative).  
  Reviews have been preprocessed and encoded as sequences of word indices (integers).  
- **Context:**  
  Sentiment analysis helps understand public opinion and user satisfaction in domains such as entertainment, marketing, and product analysis.  
- **Size:**  
  - 25,000 reviews for training  
  - 25,000 reviews for testing  
- **Labels:**  
  - `0` → Negative  
  - `1` → Positive  

---

## 🧠 Model Description
The model is built using **TensorFlow/Keras** and employs a **SimpleRNN** architecture with:
- Word **embedding layer**
- **Recurrent** layer (RNN) to capture sequence context
- **Dense** layers with **Dropout** and **L2 regularization** to prevent overfitting
- **Sigmoid activation** for binary classification

**Optimization:** Adam  
**Loss Function:** Binary Cross-Entropy  
**Metrics:** Accuracy, Precision, Recall, F1-Score  

---

## 🚀 Key Features
- Text preprocessing and tokenization using Keras
- Regularization techniques (Dropout, L2) for better generalization
- Threshold tuning for optimal F1-Score
- Model checkpointing and early stopping
- Visualization of training accuracy and loss curves
